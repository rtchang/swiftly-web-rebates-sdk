Setup Instructions

To run this, include the following script in the head <script src="https://swiftly-web-rebates-sdk.vercel.app/js/rebates-web-sdk.js"></script> and in the body call this <script type="text/javascript"> window.initializeSwiftly('b0fbc8a0-c305-4adc-97c0-80d7548ea2f9') </script> to call a single rebate (rebate id: 75d0f567-e4c1-457f-a32a-b61bff87e211) <swiftly-rebates id="75d0f567-e4c1-457f-a32a-b61bff87e211"></swiftly-rebates> to call for a list of rebates (category: Coupon/sm-beer-wine-spirits): <swiftly-rebates category="Coupon/sm-beer-wine-spirits"></swiftly-rebates>
